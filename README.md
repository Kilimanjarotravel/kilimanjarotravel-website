# Kilimanjaro Travel Website

Modern website built with Next.js + Tailwind CSS.

## Run locally

```bash
npm install
npm run dev
```

Open: http://localhost:3000

## Pages included
- Home
- About
- Safaris
- Kilimanjaro Treks
- Zanzibar
- Car Hire
- Contact

Brand: Kilimanjaro Travel — See More. Feel More.
